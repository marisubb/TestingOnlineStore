# (В разработке) Проект по тестированию интернет-магазина [МУЗТОРГ](https://www.muztorg.ru/)

### 📌 Описание проекта
Целью данной работы является функциональное и нефункциональное тестирование веб-приложения интернет-магазина [МУЗТОРГ](https://www.muztorg.ru/). В рамках проекта был проведен анализ основных пользовательских сценариев, проверка критических функций и составление отчетов о дефектах.

### 🎯 Объект тестирования
*   **Название:** МУЗТОРГ
*   **Тип приложения:** Интернет-магазин (Web)
*   **Ссылка:** https://www.muztorg.ru/

### 📝 Что было протестировано
В ходе ручного тестирования были проверены следующие модули:

1. **Главная страница:**
    *   Header(верхняя часть страницы).
    *   Основная рабочая область.
    *   Footer(нижняя часть веб-страницы).
3. **Каталог товаров и поиск:**
    *   Проверка работы фильтров (по цене, бренду, атрибутам).
    *   Функция поиска (позитивные и негативные сценарии).
    *   Сортировка результатов.
4.  **Карточка товара:**
    *   Добавление в корзину / в избранное.
    *   Проверка отображения цен (старая/новая, скидки).
    *   Выбор опций товара.
5.  **Корзина:**
    *   Редактирование корзины.
    *   Применение промокодов.
6. **Оформление заказа**
    *   Ввод данных доставки и выбор способа оплаты.
    *   Подтверждение заказа.
8.  **Модуль авторизации и регистрации:**
    *   Создание нового аккаунта.
    *   Вход по логину/паролю.
    *   Восстановление пароля.
9.   **Личный кабинет:** 
    *   Просмотр истории заказов.
    *   Редактирование персональных данных.
10.  **Тестирование верстки (UI):**
    *   Проверка адаптивности для мобильных устройств.
    *   Кроссбраузерное тестирование.
